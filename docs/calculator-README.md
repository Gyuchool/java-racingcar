# 문자열 덧셈 계산기😎

## 기능 구현 목록 👋

- [x] 문자열 또는 null 값을 입력할 경우 `0`을 반환해야 한다.(예 : “” => 0, null => 0)
- [ ] 숫자 하나를 문자열로 입력할 경우 해당 숫자를 반환한다.(예 : “1”)
- [ ] 쉼표 또는 콜론으로 숫자를 나눈다.
- [ ] `//`와 `\n` 문자 사이에 커스텀 구분자를 지정할 수 있고 해당 구분자로 숫자를 나눈다. (“//;\n1;2;3” => 6)
- [ ] 구분자로 구별된 숫자의 합을 반환한다.

## 예외 처리 사항

- [ ] 음수나 문자가 추출된 경우 RuntimeException 예외가 발생해야 한다.
